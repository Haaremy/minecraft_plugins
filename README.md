# Mc.Haaremy.de Plugins
Plugins for Velocity and PaperMc Server.

## My Setup:
```bash
├── Velocity
|   ├── server.jar
|   ├── plugins
|   |   ├── hmyVelocity.jar
|   ├── Servers
|   |   ├── Paper (Lobby)
|   |   |   ├── plugins
|   |   |   |   ├── hmyPaper.jar
|   |   ├── Paper (Sonstige)
|   |   |   ├── plugins
|   |   |   |   ├── hmyPaper.jar
```

## hmyVelocity

| :Command: | :Description: | :Permission: |
|:---------|:-------------|------------:|
|lobby| Brings you back to Lobby.||
|hmy server [name]| Brings you to a Server, defined in velocity.toml|hmy.server.[name]|
|         |             |            |
